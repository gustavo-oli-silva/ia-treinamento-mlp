# Nota sobre os Notebooks
 
Todos os notebooks entregues nesta atividade — `MLP_Pequena`, `MLP_Média`, `MLP_Grande` e `MLP_Testes_de_Funções` — são baseados no **mesmo modelo base**, compartilhando a mesma estrutura de pré-processamento, loop de treinamento e visualização de curvas.
 
As variações entre eles servem **apenas para explicitar os parâmetros** de cada configuração testada, como o número de neurônios por camada e a função de ativação utilizada. O objetivo é tornar cada configuração clara e isolada, facilitando a comparação dos resultados apresentados no relatório.
